# Checklist

Use these prompts, not as a strict gate.

- What is the real component here?
- Are we using it correctly?
- Is this actually a CSS / flex / radius problem?
- Is this control too custom for PS UXP?
- Is there a sibling misuse elsewhere?
- If I change this, should Chrome change too?
- Did this conversation reveal a stable new pitfall worth adding back to this skill?
