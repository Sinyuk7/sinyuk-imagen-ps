# Provider OpenAPI Reference Archive

This directory stores provider endpoint reference snapshots used when the
provider image-endpoint contract was shaped.

These files are reference material, not current product roadmap or active Loop
authority. Verify important endpoint behavior against current code, tests,
upstream provider documentation, or approved live smoke before changing provider
semantics.

Current provider authority lives in:

- `packages/providers/AGENTS.md`
- `docs/ENGINEERING_CONTEXT.md`
- `docs/TESTING.md`
- provider implementation and tests under `packages/providers/`
