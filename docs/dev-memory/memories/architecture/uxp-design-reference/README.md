# UXP Design Reference Archive

This directory archives historical UXP design notes and static UI prototypes.
It is project memory, not current implementation authority.

Current app authority lives in:

- `apps/app/AGENTS.md`
- `apps/app/SPEC.md`
- `apps/app/STATUS.md`
- `docs/TESTING.md`

Use these files only as historical design context when a current task explicitly
needs them. Do not treat old visual drafts, old phase wording, or prototype
behavior as implemented app contract.

The original `apps/app/dcos/` directory name was a misspelling of `docs`; the
content has been moved here to keep historical context out of the executable
app surface.
