# AGENTS.md

## Project Overview

`@imagen-ps/core-engine` — Host-agnostic runtime 层。负责 job lifecycle 管理、workflow 编排、provider dispatch 边界、in-memory store 与 event bus。不负责 provider 参数语义、UI 逻辑、文件系统或网络 IO。
