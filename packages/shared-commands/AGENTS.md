## Project Overview

`@imagen-ps/shared-commands` — 公共 application/use-case 层。负责 command facade、runtime assembly、CommandResult 错误包装与 adapter 注入。不负责 UI 渲染、CLI 解析、Photoshop/UXP host 适配及任何 surface-specific 逻辑。
